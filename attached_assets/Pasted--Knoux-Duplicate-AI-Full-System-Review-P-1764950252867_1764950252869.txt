6️⃣ **Backend & Modules**
- Confirm File Scanner module works for all supported file types.
- Verify Duplicate Detection Engine identifies duplicates correctly.
- Test Smart Suggestion Engine selects best versions correctly.
- Ensure Safe Trash module operates offline.
- Confirm AI Summarizer produces correct document summaries.
- Verify Version Detection correctly identifies older and newer versions.

7️⃣ **Full System Testing**
- Test end-to-end workflow:
  1. Open application → Splash → Dashboard
  2. Select folders and scan
  3. Review duplicates and smart suggestions
  4. Apply rules / delete / move to trash

- Ensure all statistics (Total Files, Duplicates Found, Storage Efficiency) update correctly in Arabic.
